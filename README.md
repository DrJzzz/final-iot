# Monitor de condiciones ambientales para cuarto de servidores.

El contenido de los files es el siguiente:

`monitor.py` : Contiene el script principal que es ejecutado por el servicio en la raspberry pi
`monitor.service` : Contiene la configuracion del servicio que ejecuta el script
`requirements.txt` : Requermientos de librerias de python 
`sensorTest.py` : Pequeño script para probar el sensor
`webApp.py` : Web App que recibe las API calls del monitor
